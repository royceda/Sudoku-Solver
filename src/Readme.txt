Projet Sudoku

Auteurs :
Jérémy Arnault-Macabiau
Julien Marzin
Pierre Gaulon

Compiler le programme :

make

Nettoyer le répertoire :

make clean

Exécuter le programme :

./sudoku <nomDeFichierTxt>
(On peut aussi faire ./sudoku, le nom de fichier sera demandé par le programme)

Jeux à disposition : comparison.txt, jigsaw.txt, killer.txt, killer4.txt, sudoku.txt, sudoku4.txt, sudoku16.txt
